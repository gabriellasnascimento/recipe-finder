# Mini Project #3 Proposal: API Recipe Finder

## What I'm building:
I am building a recipe finder app where users can search for meals and see recipe ideas with ingredients and instructions.

## Which API I'm using:
I am using TheMealDB API: https://www.themealdb.com/api.php

## Why I chose this:
I chose this API because I like food and cooking, and the data is useful because users can search for real recipes instead of seeing hard-coded information.

## Core features:
- Search for recipes by meal name
- Display recipe cards with the meal name, image, and category
- Click on a recipe to see more details, including ingredients and instructions
- Show a loading message while the API is fetching data
- Show a friendly error or “no results found” message if nothing comes up

## What I don't know yet:
I am still learning how to use fetch(), async/await, and JSON. I also need to better understand how to take the API response and display only the information I need on the page.
